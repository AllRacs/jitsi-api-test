# Jitsi Meet API Test

Testing Jitsi Meet API for embed call/videocall system.

## Requirements

- NPM and NodeJS

## How to run

1. Clone repo
```bash
$ git clone https://github.com/AllRacs/jitsi-api-test.git
```
2. Install dependencies
```bash
$ npm install
```
3. Run project
```bash
$ npm run dev
```
4. Browser search
	- localhost:8000
